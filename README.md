A software that supports vector data to 3DTiles, oblique photogrammetry (osgb) model to 3DTiles, oblique model clipping, point cloud to 3DTiles, image tiling, and terrain tiling.

Small in size, fast in processing.

Provides HTTP API, along with a built-in comprehensive WebGUI interface.

Official Website：https://v3dtiles.top

User Manual: [Feishu Docs - V3DTILES Software User Manual](https://v3dtiles.feishu.cn/docx/AaVtd5PjpoikP2xoV73cnM5qnue)


## Main Features

+ Tiling Project Management
  + Convert vector data (shp, gdb, gpkg, pg, etc.) to 3DTiles, mainly for generating building white models
  + Convert oblique photogrammetry (osgb) data to 3DTiles
  + Oblique model clipping (precise clipping by polygon extent)
  + Point cloud to 3DTiles
  + Tile images in XYZ or TMS format
  + Terrain data tiling (quantized-mesh-1.0)
+ Tiling Task Management
  + Running tasks
  + Completed tasks
+ Statistical Analysis
+ Service Distribution
  + Single file/folder
  + 3DTiles
  + Terrain service
  + XYZ/TMS image service
+ 3D Visualization
  + Layer management
  + Scene management
  + Place name search
  + Measurement tools
  + Weather simulation
  + Scene lighting
  + Terrain processing
